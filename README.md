# Beyond the New Keynesian Channel: Fiscal Shocks and Inflation in the Euro Area

This repository contains the code and supplementary material for the research project **Beyond the New Keynesian Channel: Fiscal Shocks and Inflation in the Euro Area**.

## Overview

This project investigates whether standard New Keynesian transmission mechanisms fully explain the inflationary effects of fiscal shocks in the Euro Area.

Fiscal innovations are identified using a two-way fixed-effects first-stage regression, while dynamic responses are estimated through Local Projections. The analysis also explores cross-country heterogeneity, sovereign risk, fiscal regimes and inflation expectations.

## Methods

- Local Projections
- Panel Data Econometrics
- Two-Way Fixed Effects
- Robustness Analysis
- Inflation Expectations
- Macroeconomic Forecasting
